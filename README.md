# Player-stats-analyser
A python project that uses pandas to analyse football player data, calculate averages, and identify the top scorer 
## what it does
- stores football player stats in a dataset
- displays the dataset as a table
- calculates average goals scored
- saves ad reads data from CSV file

## skills used
- python fundamentals( loops, lists, dictionaries)
- data analysis using **pandas**
- file handling(CSV read/write)
